# Rota Viva

> Uma solução inteligente, sustentável e moderna para conectar caminhos e transformar a mobilidade urbana no seu dia a dia.

---

## Sobre o Projeto

O Rota Viva é um projeto desenvolvido como TCC. Nosso objetivo é criar uma plataforma web intuitiva que facilite o acesso a serviços essenciais de viagem e transporte, centralizando informações sobre aviação, compra de ingressos, locação de veículos e reservas de hotéis em uma única interface amigável.
